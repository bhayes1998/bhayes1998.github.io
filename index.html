Projects

Below you can find some projects that I am particularly proud of. Some of these are assignments I did during ym time at Miami, and some of them are personal projects. You can also find a link to the Github project page or Google Drive for each one.

Loan Shark [2020-2021]

This was my senior capstone project at Miami that I worked on with a group of three other people. It is an Android mobile app designed to keep track of items that you have borrowed from someone or have lent out to someone, with integrated Google account support. It was written in Dart using the Flutter framework, with Google Firebase as our database solution. [Note: this project is owned by Miami University, so I am not permitted to share the source code. I have linked to the final project video instead.]

Toilet Talk [2019]

The final project for my databases class. This is a web app built from the ASP.NET framework that acts as a review site for all of the bathrooms on Miami Universities campus. Users could leave short reviews and bathrooms as well as give them a rating. Each bathroom page contained some info about the bathroom such as number of sinks/stalls, as well as the allowed gender. Reviews were stored in a MySQL database.

Running App [2020]

My final project for my mobile apps class. It is a basic running app that can store the users run time, distance, and date. After a run is added, it can be updated or deleted, with each run being stored in a SQLite database. Users can also see their yearly, monthly, and weekly totals visually represented in a bar graph, as well as calculate their age grade for a set of distances.

Unity Projects [2020]

A collection of the projects I did for my game design class. I couldn't decide which one I liked the best, so I've included all of them. The labs are mostly one-off learning exercises, but the three projects show off more of what I was capable of in Unity. The first project was a carnival-style shooter in the form of naval warfare, the second project is a katamari game, and the thrid is a basic capture the flag shooter. All of these projects were created in Unity using C# for scripting.

Basic Command Shell [2019]

A C++ project created for my systems class that creates a Linux command terminal and can perform basic Linux commands. It involves forking and manipulating child processes using the execvp() function. The given commands can be done serial or in parallel.

Z+- Interpreter [2020]

This was a fun Java project that I did for my comparative programming languages class. It involves a made-up basic programming language called Z+-, which can do very simple math as well as single and nested for loops. The program reads in a text file containg the Z+- code and parses through it to decide what to do.
